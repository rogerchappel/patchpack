# Roadmap

## 0.1 MVP

- deterministic `.ppack` bundles
- create, inspect, and apply commands
- dry-run apply by default
- fixture-backed smoke tests
- conservative secret and path checks

## Next

- richer binary patch refusal/metadata
- optional command runner for recorded validation commands
- signed bundle metadata
- stricter schema validation and migration tests
- editor-friendly inspect output

## Non-goals

- hosted service
- telemetry
- account-based sharing
- hiding the underlying git patch format
